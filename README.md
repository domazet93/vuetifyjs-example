# vuetify

> A Vue.js and Vuetify simple example

## Prerequisites

* Clone or download repo
* Node.js

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
